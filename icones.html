Bonus

Icônes Lucide

Blocks Master embarque en local la bibliothèque Lucide Icons (v1.8.0+). Plus de 1400 icônes vectorielles propres et cohérentes, disponibles dans tous vos blocs sans aucune configuration.

Syntaxe de base

Pour afficher une icône, utilisez la balise <i> avec l'attribut data-lucide :

<i data-lucide="nom-de-licone"></i>

Le script Lucide inclus dans le plugin scanne la page et remplace automatiquement ces balises par des SVG vectoriels optimisés.

Combiner Lucide et Tailwind CSS

Puisque Lucide génère du SVG, vous pouvez utiliser les classes Tailwind pour contrôler la taille (tw-w-X tw-h-X) et la couleur (tw-text-*) :

<!-- Icône small, couleur indigo -->
<i data-lucide="arrow-right" class="tw-w-4 tw-h-4 tw-text-indigo-600"></i>

<!-- Icône large, couleur personnalisée -->
<i data-lucide="sparkles" class="tw-w-10 tw-h-10 tw-text-violet-500"></i>

Exemple : Carte Feature avec icône

<div class="tw-flex tw-items-start tw-gap-4 tw-p-6 tw-bg-white tw-rounded-2xl tw-shadow-sm">

    <div class="tw-flex-shrink-0 tw-w-12 tw-h-12 tw-bg-indigo-100 tw-rounded-xl
                tw-flex tw-items-center tw-justify-center">
        <i data-lucide="shield-check" class="tw-w-6 tw-h-6 tw-text-indigo-600"></i>
    </div>

    <div>
        <h3 class="tw-font-semibold tw-text-slate-900">Sécurité maximale</h3>
        <p class="tw-text-slate-500 tw-text-sm tw-mt-1">
            Protégé par nonces WordPress sur chaque action.
        </p>
    </div>

</div>

Catalogue des icônes

Pour trouver le nom d'une icône, consultez le catalogue officiel en ligne. Copiez simplement le petit nom en minuscules affiché sous l'icône (ex: arrow-right, user-circle, mail).
