Blocks Master / SiteMaster

La suite WordPress pour développeurs qui veulent créer des sites sur-mesure, rapides et maintenables — sans lourdeur, sans builders, sans compromis.

Le Lab

Créez des blocs Gutenberg en PHP pur avec Tailwind CSS. Versioning intégré, attributs éditables et IDE dans le navigateur.

SiteMaster

Gérez des structures de données complexes et liez-les à des pages modèles dynamiques avec routage automatique par URL.

Formulaires Headless

Moteur de formulaires complet — validation, stockage DB, emails — avec liberté totale sur le markup HTML de votre côté.

Support Elementor

Chaque bloc créé dans le Lab est automatiquement disponible comme widget Elementor dans la catégorie "SiteMaster", avec ses attributs.

La philosophie

⚡

PHP-First

Vous écrivez du PHP et du HTML. Le plugin s'occupe de l'enregistrement Gutenberg. Pas de build, pas de Node, pas de React.

🚀

Performance Native

Architecture hybride : CPT WordPress pour le contenu, tables SQL optimisées pour la logique système, fichiers physiques pour le rendu rapide.

🛠️

Developer Experience (DX)

IDE intégré, Tailwind CSS embarqué, API PHP granulaire — tout est conçu pour que vous soyez productif en moins de 5 minutes.

Architecture v3.0

Données Stockage Raison
Blocs (code) wp_blocks_master Versioning, accès rapide, historique
Formulaires wp_sm_forms Tables optimisées, requêtes ultra-rapides
Soumissions wp_sm_form_entries JSON par entrée, indexé par form_id
Items (contenu) wp_posts CPT natif — compatible SEO, cache, plugins

Pré-requis

  • ✓ WordPress 6.0+
  • ✓ PHP 7.4+ (8.1+ recommandé)
  • ✓ Éditeur Gutenberg actif
  • ✓ Aucune dépendance externe — Tailwind CSS embarqué en local
